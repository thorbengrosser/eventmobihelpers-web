Welcome to the EventMobi Helpers Web App

Please choose one of the following options to proceed:

  • Delete Sessions in Group

    This feature allows you to delete sessions associated with a specific track within an event. You can select an event and track, and all sessions linked to that track will be removed.

  • Add People to Group by Email

    Use this feature to add people to a specified group based on their email addresses. You can paste a list of email addresses, and the tool will assign those users to the group you select.

  • Manage Chat Settings

    This tool lets you enable or disable chat functionality for users in a specific group within an event. Select the event and group, and then choose whether to enable or disable chat for all members of that group.

  • Mass Delete Sessions

    This feature allows you to delete multiple sessions by entering their session IDs. Simply select an event, paste a list of session IDs, and the tool will remove all listed sessions.

  • Expert Session Editor

    Manually edit session information, including chat settings, AAQ settings, and more. Select an event and session, then make your desired changes.
